# WinterCabin
三维图形程序设计课程:利用opengl搭建场景  
库:  (需要电脑配置)glew freeglut glfw3  
     (代码中已存在)glm,stb_image  
场景构成:房子，树，雪人，路灯，雪地  
动画:光线变化，白天与黑夜的效果  
操作方式:  
  w/a/s/d:视角变换(摄像机位置)  
  鼠标右键:视角变换(摄像机位置)  
  鼠标左键:视角变换(摄像机方向)
